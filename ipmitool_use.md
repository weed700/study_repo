# ipmitool useage

* 유저 확인
 * ipmitool user list 1
* 비밀 번호 변경
 * ipmitool user set passwd <user id> <password>
* 확인 방법 다른 노드에서
 * ipmitool -I lanplus -H <IPMI IP> -U <user> -P <password> chassis status
